# ShopVista – E-commerce Frontend Website

ShopVista is a frontend-based e-commerce website developed using HTML and CSS. The project focuses on creating a clean, user-friendly interface for browsing products and simulating an online shopping experience.

---

## Features

* Home page with featured products and banners
* Product listing and detailed product pages
* Individual product view section
* Easy navigation across pages
* Basic responsive design for different screen sizes
* Simple and intuitive UI/UX

---

## Project Structure

* `index.html` – Home page
* `shop.html` – Product listing page
* `sproduct.html` – Single product details page
* `blog.html` - Blog about the website.
* `cart.html` – Shopping cart page *(basic implementation)*
* `contact.html` – Contact form page
* `y.css` – Main stylesheet for the entire website

---

## Technologies Used

* **HTML5** – Structure of the website
* **CSS3** – Styling and layout design
* **JavaScript** *(basic usage)* – UI interactivity (if applicable)
